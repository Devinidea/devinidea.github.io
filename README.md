# Devin Zhao — Portfolio Website

Personal portfolio website for DevOps & Cloud Engineer.

## 🚀 Quick Start

### Local Development
```bash
# Simply open in browser
open index.html
```

### Deploy on GitHub Pages (free)

1. Create a new GitHub repository named: `/devinidea.github.io`
2. Push this folder to the `main` branch
3. Done! Your site will be live at: ` https://devinidea.github.io/`

## 📁 File Structure

```
portfolio/
├── index.html      # Main portfolio page
├── styles.css      # All styles
└── README.md       # This file
```

## ✨ Features

- Dark theme with gradient accents
- Responsive design (mobile, tablet, desktop)
- AWS architecture code block in hero
- Timeline for work experience
- Skills grid
- Contact form (Formspree)
- GitHub Pages ready

## 🔧 Customisation

### Update contact info
Edit the `#contact` section in `index.html`.

### Update form endpoint
Replace `https://formspree.io/f/yourformid` with your [Formspree](https://formspree.io) form ID (free tier available).

### Update GitHub link
Change `https://github.com` in the contact section to your actual GitHub profile URL.

## 🎨 Color Scheme

Edit the CSS variables at the top of `styles.css`:

```css
:root {
    --accent: #3b82f6;          /* Primary blue */
    --gradient-1: #3b82f6;     /* Blue */
    --gradient-2: #8b5cf6;     /* Purple */
    --gradient-3: #06b6d4;     /* Cyan */
}
```
